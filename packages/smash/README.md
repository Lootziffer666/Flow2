# SMASH

SMASH ist die **Entblockungsschicht** des Ökosystems.

SMASH erzeugt keine Texte. Es liefert kurze, gezielte Mikrointerventionen, um aus Schreibstillstand wieder in Handlung zu kommen.

## Rolle im System

- **LOOM** klassifiziert strukturelle/zustandsnahe Signale.
- **FLOW** reduziert formale Reibung.
- **SPIN** unterstützt bei Ausdruck und Strukturarbeit.
- **SMASH** greift, wenn das Problem primär Blockade ist.

## In Scope

- kompakte Interventionsmechaniken
- schneller Einstieg
- klare Rückführung in echten Schreibprozess
- kurze, zustandsnahe Impulse statt Dauerbeschäftigung

## Out of Scope

- Casual-Game-Sammlung
- Schreibkurs-Ersatz
- Prompt-/Ghostwriting-Maschine
- generative Texterzeugung
