# Rules Documentation Status

This directory contains FLOW rule/reference documentation with different purposes.

## Canonicality

- **Canonical current baseline:** `Aktuellstes_Regelwerk.md`
  - Use this as the primary maintained reference for current rule behavior and decisions.
  - This baseline should preserve FLOW’s broad usability while keeping its LRS/Dyslexia-prioritized intent explicit.

- **Extended/background companion:** `Erweitertes_Regelwerk.md`
  - Contains broader rationale, matrices, and longer-form background context.
  - Not the primary file for "current baseline" decisions.
  - Intentional relationship: extension/supporting layer, not a competing canonical ruleset.

- **Source/reference link catalog:** `Links_Quellen_NEU_DE.md`
  - Maintained bibliography/reference list.

- **English link companion:** `Links_Quellen_NEU_EN.md`
  - English-oriented reference companion.

## Migration note

Legacy concept material from `old_main/concepts/` has been archived under `docs/archive/`.
When updating rule behavior, keep `Aktuellstes_Regelwerk.md` aligned first, then sync supporting context docs as needed.
